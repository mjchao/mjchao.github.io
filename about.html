About Me

I started programming in 2008 and since then, I have written an enormous amount of code and gained practical experience from numerous projects. Currently, I work in the algorithmic trading industry and contribute to low-latency software that executes orders on approximately one hundred exchanges.

My biggest focus is on the software development process. I emphasize requirements and design to reduce the amount of time wasted on writing code that ends up being rewritten. I generate timelines and estimates to maintain visibility on progress. I write plenty of documentation so that everyone can get a high-level understanding of my projects. And I create examples and tutorials so that anyone else can start using or contributing to my projects. The extra effort pays off in the long run when people are able to collaborate efficiently in a massive codebase.
